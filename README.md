# qclient-web

a modern web interface for the qclient dashboard, built with react, typescript, vite and vanilla css.

## overview

a premium, high-performance web landing page for a modern minecraft client dashboard, built to showcase premium frontend design and fluid web interactions. the entire interface features a sleek dark aesthetic, customized layout structures, and smooth micro-animations.

## features

* **floating background particles**: a dynamic canvas that renders slowly rising soft square pixel particles to capture the essence of game portals, optimized for smooth rendering without performance drops.
* **shimmering action buttons**: borderless interactive buttons with custom diagonal shimmer sweeps and active scaling click feedback.
* **sticky footer architecture**: a responsive layout where social links, copyright declarations, and legal disclaimers are placed cleanly and only appear at the bottom of the document scroll.
* **modular subviews**: fully responsive subview modules for service status, support desks, terms sheets, and faq accordions with borderless panels and custom slide-up entry animations.

## stack

* **framework**: react (with typescript)
* **bundler**: vite
* **styling**: vanilla css
* **typography**: sf pro display (thin font weights)

## scripts

first, install all project dependencies:

```bash
npm install
```

to launch the local dev server:

```bash
npm run dev
```

to compile the application for production:

```bash
npm run build
```
